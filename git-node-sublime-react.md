##### Github配置

> 将ssh key串添加到github

> 生成 github 的 ssh key

```
ssh-keygen -t rsa -C "xx@xx.com"
```

> 建议提示都不要输入任何密码, 直接回车

```
cat  ~/.ssh/id_rsa.pub
```

> 复制生成的key到github

> 在github, 右上角 Profile Settings ->  Github 的 SSH Keys 管理台中添加
